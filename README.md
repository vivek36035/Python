# COVID-19 Data Analysis Dashboard

## Overview
A live dashboard built using **Python + Streamlit** to visualize global and country-wise COVID-19 data from [covid19api.com](https://covid19api.com).

## Features
- Live COVID-19 statistics by country.
- Interactive bar and pie charts.
- Global summary metrics.
- Clean Streamlit UI.

## Run Instructions
```bash
git clone https://github.com/yourusername/covid19_dashboard.git
cd covid19_dashboard
pip install -r requirements.txt
streamlit run app.py
